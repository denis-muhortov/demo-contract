## Setup

Убедитесь в том, что установлены зависимости:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

Запустите сервер разработки на `http://localhost:3000`

```bash
npm run dev
```

## Production

Создайте приложение для production:

```bash
npm run build
```

Локальный предварительный просмотр production сборки:

```bash
npm run preview
```
